# DUROXO Smart Lithium Battery System

## Overview

DUROXO is a smart **LiFePO₄ battery management system (BMS)** developed for **forklifts and material handling equipment** in collaboration with **Battery Asia (S) Pte Ltd** and **Enterprise Singapore**. The project transformed a laboratory innovation into a commercially deployed product that is now operational across **7 countries in Asia**.

The project was honoured with the **Public Sector Transformation Award 2023 (Dare to Do)** by the Singapore Government.

<!-- --- -->

## Key Features

- **Active cell balancing** — ensures uniform charge distribution across all cells for extended battery life
- **SOC/SOH estimation** — real-time state-of-charge and state-of-health monitoring using advanced algorithms
- **Fault diagnostics** — intelligent fault detection and isolation for safe operation
- **4G/WiFi IoT connectivity** — remote monitoring via a web-based platform
- **VDE-certified** — meets IEC 62619:2017 international safety standards
